# CIS-11 – Homework 5: The Largest Number

**Student:** Tu Nombre (Abraham Pairo)
**Course:** RCC CIS/CSC-11 – Fall 2025
**Assignment:** The Largest Number

This project contains:
- `largest.cpp` – C++ driver
- `manager.asm` – Assembly coordinator (allocates stack array, calls modules)
- `input_array.cpp` – C++ input routine
- `output_array.asm` – Assembly printer
- `find_largest.asm` – Assembly function returning index of largest
- `Makefile` – builds `main`

All integers and counts are 64-bit signed (C++ `long` / ASM qword).
Executable name: `main`.
